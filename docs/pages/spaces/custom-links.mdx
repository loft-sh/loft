Loft lets you add custom links to space instances, for example to point to a github pull request or jira issue.

:::tip Virtual Clusters Too!
Check out the Virtual Clusters [Advanced Topics ](../virtual-clusters/custom-links) for custom links.
:::

    <TabItem value="ui">
      <Step>

      </Step>
      <Step>
        Edit an existing Space or create a new one
      </Step>
      <Step>
        Click <Label>Add Link</Label> to add a new link or <Label>Change</Label> to edit existing ones
      </Step>
      <Step>
        Enter a URL and optionally a label for your link
      </Step>
      <Step>
        Save your links by clicking <Label>Save</Label>
      </Step>
      <Step>
        Click on the <Button>Create Space</Button>, 
        or <Button>Save Changes</Button> if you're editing an existing space,
        to apply all changes
      </Step>
    </TabItem>

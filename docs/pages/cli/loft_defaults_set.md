Set default value

## Synopsis


```
loft defaults set [flags]
```

```
#######################################################
################## loft defaults set ##################
#######################################################
Set sets a default value for lofctl
loft defaults set $KEY $VALUE

Example:
loft defaults set project your-project

Supported keys include:
project
#######################################################
```


## Flags

```
  -h, --help   help for set
```

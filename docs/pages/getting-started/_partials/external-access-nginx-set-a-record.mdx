[Set up a DNS A record](https://www.google.com/search?q=set+up+a+record+dns) to the ingress
address (x.x.x.x). Make sure Loft is reachable at the address, you can check that with curl as
shown below, or simply browse to the address in your browser of choice:

```bash
curl https://loft.mydomain.tld/version --insecure | jq
{
  "kind": "Version",
  "apiVersion": "version.loft.sh",
  "metadata": {
    "creationTimestamp": null
  },
  "version": "v3.0.0",
  "major": "3",
  "minor": "0",
  "instance": "",
  "kubeVersion": "v1.24.2",
  "newerVersion": "",
  "shouldUpgrade": false
}
```

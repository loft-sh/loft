The SAML 2.0 connector in dex does **not** support refresh tokens "since the SAML 2.0 protocol doesn't provide a way to requery a provider without interaction" (see [dex documentation for SAML 2.0](https://dexidp.io/docs/connectors/saml/)).
:::

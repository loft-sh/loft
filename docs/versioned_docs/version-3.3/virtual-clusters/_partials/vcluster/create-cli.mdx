When creating a virtual cluster from the Loft CLI, you will need to provide the name of the
project in which to deploy the virtual cluster, and the name of template to use:

```bash
loft create vcluster [vcluster-name] --project [project-name] --template [template-name]
```

:::info Template
If you omit specifying a template, Loft will automatically select one for you (if there is a
default template in the project specified) or prompt you to select a template.
:::

:::info Kube-Context
Running `loft create vcluster` will automatically add a kube-context to your kube-config file,
so you can immediately run `kubectl` commands right after creating a virtual cluster.
:::

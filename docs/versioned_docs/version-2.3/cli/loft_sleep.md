Put a space to sleep

## Synopsis


```
loft sleep [flags]
```

```
#######################################################
###################### loft sleep #####################
#######################################################
Sleep puts a space to sleep.

Example:
loft sleep myspace
loft sleep myspace --cluster mycluster
#######################################################
```


## Flags

```
      --cluster string   The cluster to use
  -h, --help             help for sleep
```

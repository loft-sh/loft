Wakes up a space

## Synopsis


```
loft wakeup [flags]
```

```
#######################################################
###################### loft wakeup ####################
#######################################################
Resumes a sleeping space.

Example:
loft wakeup myspace
loft wakeup myspace --cluster mycluster
#######################################################
```


## Flags

```
      --cluster string   The cluster to use
  -h, --help             help for wakeup
```

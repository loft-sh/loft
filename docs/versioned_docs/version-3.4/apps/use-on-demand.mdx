Apps can be deployed "on-demand" into any cluster, space, or virtual cluster managed by Loft.
This can be a useful tool for adding pre-defined applications to newly connected clusters, or to
newly created spaces or virtual clusters that are not already using a template that includes Apps.

<Tabs
    defaultValue="cluster"
    values={[
        {label: '(Physical) Cluster', value: 'cluster'},
        {label: 'Space', value: 'space'},
        {label: 'Virtual Cluster', value: 'vcluster'},
    ]}>
    <TabItem value="cluster">
        <Flow id="add-app-cluster">
            <Step>
                Select the <NavStep>Clusters</NavStep> field on the left menu bar.
            </Step>
            <Step>
                From the Clusters sub-menu, select the Clusters option.
            </Step>
            <Step>
                Find the cluster you would like to deploy an App to in the list of connected
                clusters and click on that cluster.
            </Step>
            <Step>
                In the Cluster Management view, click the <Button>Apps</Button> button.
            </Step>
            <Step>
                Select the App you would like to install from the App thumbnails, or, if there
                are no recommend Apps, click the <Button>Install App</Button> and find  the
                desired App in the <Label>Select an App</Label> drop down menu.
            </Step>
            <Step>
                In the Install App screen enter the namespace to install the App to in the
                <Label>Namespace</Label> field.
            </Step>
            <Step>
                Enter any other required parameters for your App -- this may vary depending on
                the type of App and whether or not the App creator has specified parameters.
            </Step>
            <Step>
                Click the <Button>Install</Button> button.
            </Step>
        </Flow>
    </TabItem>
    <TabItem value="space">
        <Flow id="add-app-space">
            <Step>
                Select the <NavStep>Projects</NavStep> field on the left menu bar.
            </Step>
            <Step>
                Select the project containing the space that you'd like to install the App in
                from the Project drop down menu.
            </Step>
            <Step>
                From the Projects sub-menu, select the Spaces option.
            </Step>
            <Step>
                Find the space you would like to deploy an App to in the list of connected
                clusters and click on that cluster.
            </Step>
            <Step>
                In the Space Management view, click the <Button>Apps</Button> button.
            </Step>
            <Step>
                Select the App you would like to install from the App thumbnails, or, if there
                are no recommend Apps, click the <Button>Install App</Button> and find  the
                desired App in the <Label>Select an App</Label> drop down menu.
            </Step>
            <Step>
                Enter any other required parameters for your App -- this may vary depending on
                the type of App and whether or not the App creator has specified parameters.
            </Step>
            <Step>
                Click the <Button>Install</Button> button.
            </Step>
        </Flow>
    </TabItem>
    <TabItem value="vcluster">
        <Flow id="add-app-vcluster">
            <PartialVirtualClusterInstallApp/>
        </Flow>
    </TabItem>
</Tabs>

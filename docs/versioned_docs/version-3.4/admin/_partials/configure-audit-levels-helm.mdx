

Create a new file `values.yaml`:
```yaml
config:
  audit:
    enabled: true
    level: 1
```

Then apply these values with helm (make sure to specify the correct Loft version):
```
helm upgrade loft loft --namespace loft \
                       --repo https://charts.loft.sh \
                       --version $LOFT_VERSION \
                       --reuse-values \
                       --values values.yaml
```

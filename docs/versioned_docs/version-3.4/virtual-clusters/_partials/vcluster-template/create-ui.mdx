<Flow id="create-vcluster-template-ui">
  <Step>
    Go to the <NavStep>Virtual Clusters</NavStep> view using the menu on the left
  </Step>
  <Step>
    Switch to the <Label>Virtual Cluster Templates</Label> tab
  </Step>
  <Step>
    Click the <Button>Create Virtual Cluster Template</Button> button to create a new virtual cluster template
  </Step>
  <Step>
    In the drawer that appears on the right, use the field <Label>Display Name</Label> to specify a <Input>Name</Input> for your virtual cluster template
  </Step>
  <Step>
    Specify any of these options: labels, annotations, virtual cluster version, helm values and apps to deployed
  </Step>
  <Step>
    Expand the <Expander>Space Template</Expander> section to specify which <Input>Space Template</Input> should be applied for the underlying host namespace if being created as part of the virtual clusters creation process
  </Step>
  <Step>
    On the very bottom, click on the <Button>Create</Button> button to create this virtual cluster template
  </Step>

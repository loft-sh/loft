You can connect a new cluster through this API, compared to the cluster api, this will also create the necessary secret and cluster access object within the newly connected cluster.

## Example Cluster Connect

An example Cluster Connect:
```yaml
apiVersion: management.loft.sh/v1
kind: ClusterConnect
metadata:
  creationTimestamp: null
  name: my-cluster
spec:
  adminUser: my-user
  clusterTemplate:
    metadata:
      creationTimestamp: null
    spec:
      config: {}
    status: {}
  config: |-
    apiVersion: v1
    kind: Config
    clusters:
    - cluster:
    ...
status: {}

```

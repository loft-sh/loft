## Delete Project Secret
To delete a project secret follow these steps:
<Flow id="secret-delete-project-secret-data">
    <Step>Navigate to the <NavStep>Projects</NavStep> view using the menu on the left</Step>
    <Step>Select the project you'd like to configure using the drop down menu</Step>
    <Step>Click on <NavStep>Project Secrets</NavStep></Step>
    <Step>Click on the <Label>Delete</Label> button near in the drop-down list right to the name of your secret</Step>
    <Step>Click on the <Label>Delete</Label> button in the modal window to confirm the deletion</Step>
</Flow>

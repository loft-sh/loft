## Delete Global Secret
To delete a global secret follow these steps:
<Flow id="secret-delete-global-secret-data">
    <Step>Navigate to the <NavStep>Global Secrets</NavStep> view using the menu on the left</Step>
    <Step>Click on the <Label>Delete</Label> button near in the drop-down list right to the name of your secret</Step>
    <Step>Click on the <Label>Delete</Label> button in the modal window to confirm the deletion</Step>
</Flow>

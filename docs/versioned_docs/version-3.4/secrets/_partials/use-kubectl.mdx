import Flow, { Step } from '@site/src/components/Flow'


<Flow id="use-secret-kubectl">
  <Step>

Create a file named `secret.yaml` with content:
```yaml
apiVersion: v1
kind: Secret
metadata:
  name: my-secret
  labels:
    loft.sh/sharedsecret-name: my-shared-secret
    loft.sh/sharedsecret-namespace: loft
type: Opaque
```

  </Step>
  <Step>

Create the Kubernetes secret using kubectl:
```bash
kubectl apply -f secret.yaml
```

  </Step>
  <Step>

Shortly after creating the secret, Loft will populate the data of this secret with the data from our shared secret:
```bash
kubectl get secret my-secret -o yaml
```

:::tip Data Sync
By default, Loft will update the data of this secret whenever the shared secret's data is updated. To disable this behavior, set the `loft.sh/disable-sync: true` annotation on the Kubernetes secret.
:::

  </Step>
</Flow>
